- 클로저 문제
  - 스코프, 비동기, var 등이 섞여있어서 어려운 것.
  - 클로저가 문제인게 아니라, 클로저를 사용해서 해결하는 문제임.
  - 반복문과 비동기를 함께 사용하면 종종 발생

```js
function a() {
  for (var i = 0; i < 5; i++) {
    setTimeout(() => {
      console.log(i);
    }, i * 1000);
  }
}
a();
```

- 클로저
  - 함수와 외부 변수와의 관계
  -
- 그냥 실행하면 5가 5회 찍힘.
- 해결법은 var를 let으로 바꾸거나, var를 계속 쓸거라면 setTimeout 부분을 iife로 클로저를 생성해주는 것.
